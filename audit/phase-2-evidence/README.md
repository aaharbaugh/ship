# Phase 2 Evidence

Store before/after artifacts here for each category.

## Suggested contents
- `bundle/`: bundle reports, treemaps, size summaries
- `api/`: benchmark outputs, endpoint comparison tables
- `database/`: query profiles, explain analyze output, query-count comparisons
- `tests/`: coverage output, test summaries, flake triage notes
- `runtime/`: reproduction notes, screenshots, logs, recordings
- `accessibility/`: Lighthouse reports, axe outputs, contrast audit results

Every artifact should be labeled clearly enough that someone else can tell:
- what command produced it
- whether it is before or after
- what category it supports
